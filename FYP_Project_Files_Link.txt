FYP PROJECT FILES

Project Title:
Intelligent Virtual Tutor System for Early Childhood Learning Based on Deep Learning

Description:
This file provides access to the project files for the Final Year Project (FYP). The project is an AI-powered web-based virtual tutoring system designed to support early childhood learning through chatbot interaction, voice input, image classification, and real-time animal detection.
